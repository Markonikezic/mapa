1. SQL struktura tabela u bazi podataka

Kreirajte ove tabele u bazi podataka iz koje planirate da mapirate sistem u Danilovgradu:

-- Tabela sa trajnim podacima o Trafo-stanicama (TS)
CREATE TABLE trafo_stanice (
    sifra_ts VARCHAR(50) PRIMARY KEY,
    naziv VARCHAR(100) NOT NULL,
    tip_napona VARCHAR(20) DEFAULT '10/0.4 kV',
    latitude DECIMAL(9,6) NOT NULL,
    longitude DECIMAL(9,6) NOT NULL
);

-- Tabela sa mjesečnim energetskim bilansima TS
CREATE TABLE bilans_ts (
    sifra_ts VARCHAR(50) REFERENCES trafo_stanice(sifra_ts),
    obracunski_period VARCHAR(7), -- Format: 'YYYY-MM' (npr. '2026-03')
    predata_energija_kwh DECIMAL(12,2),
    preuzeta_energija_kwh DECIMAL(12,2),
    PRIMARY KEY (sifra_ts, obracunski_period)
);

-- Tabela sa trajnim podacima o potrošačima (koordinate mogu biti NULL)
CREATE TABLE potrosaci (
    sifra_potrosaca VARCHAR(50) PRIMARY KEY,
    sifra_ts VARCHAR(50) REFERENCES trafo_stanice(sifra_ts),
    ime_prezime VARCHAR(150),
    latitude DECIMAL(9,6) NULL, -- Ako nema koordinata, ostaje prazno
    longitude DECIMAL(9,6) NULL
);

-- Tabela sa mjesečnom potrošnjom po brojilima
CREATE TABLE potrosnja_potrosaca (
    sifra_potrosaca VARCHAR(50) REFERENCES potrosaci(sifra_potrosaca),
    obracunski_period VARCHAR(7), -- Format: 'YYYY-MM'
    ostvarena_potrosnja_kwh DECIMAL(10,2),
    PRIMARY KEY (sifra_potrosaca, obracunski_period)
);

2. SQL upit koji spaja podatke za Mapu (Mesečni Izvještaj)

Ovaj SQL upit koristi tvoja baza da bi za izabrani mjesec dobila sve potrebne informacije o gubicima na TS:

SELECT 
    ts.sifra_ts,
    ts.naziv AS naziv_ts,
    ts.tip_napona,
    ts.latitude,
    ts.longitude,
    b.predata_energija_kwh AS predata,
    b.preuzeta_energija_kwh AS preuzeta,
    (b.predata_energija_kwh - b.preuzeta_energija_kwh) AS gubici_kwh,
    ROUND(((b.predata_energija_kwh - b.preuzeta_energija_kwh) / b.predata_energija_kwh) * 100, 2) AS gubici_procenat,
    (SELECT COUNT(*) FROM potrosaci WHERE sifra_ts = ts.sifra_ts) AS broj_potrosaca
FROM trafo_stanice ts
JOIN bilans_ts b ON ts.sifra_ts = b.sifra_ts
WHERE b.obracunski_period = '2026-03'; -- Mijenja se dinamički iz filtera

Sledeći korak u Python-u: Koristeći biblioteku pandas i sqlalchemy, pokrećeš ovaj upit, a Pandas funkcija to_json() odmah generiše podatke u format koji vidiš ispod i crta interaktivnu mapu.
